## Installation

### Prerequisites

- Python 3.x installed  

You can check your version with:

```bash
python --version
```

### Setup

Clone the repository:

```bash
git clone https://github.com/your-username/python-guess-the-number.git
cd python-guess-the-number
```

---

## Usage

Run the script from the terminal:

```bash
python guess_the_number.py
```

The program will:

- Ask for your name  
- Generate a random number  
- Prompt you to guess the number  
- Tell you if your guess is too high or too low  
- Display the number of attempts once you win
